## Historia de Usuario - Clientes
Como administrador del sistema,
quiero poder crear, consultar, actualizar y eliminar clientes,
para tener un registro de las personas que realizan compras en la tienda.

## Historia de Usuario - Productos
Como administrador del sistema,
quiero poder gestionar el catalogo de productos (accesorios de ropa),
para mantener actualizado el inventario disponible para la venta.

## Criterios de Aceptacion

### Clientes
- Crear cliente con: nombre, documento (unico), metodo de pago, direccion
- Listar todos los clientes
- Obtener cliente por ID
- Actualizar datos de un cliente existente
- Eliminar cliente por ID
- El documento debe ser unico en la base de datos

### Productos
- Crear producto con: codigo, nombre, precio, stock, marca
- Listar todos los productos
- Obtener producto por ID
- Actualizar datos de un producto
- Eliminar producto por ID
- Los productos corresponden a accesorios de ropa

## Tareas Tecnicas
1. Crear entidad Cliente con anotaciones JPA
2. Crear entidad Producto con anotaciones JPA
3. Crear interfaces IClienteRepository e IProductoRepository
4. Crear interfaces IClienteService e IProductoService
5. Implementar ClienteService y ProductoService
6. Crear ClienteController y ProductoController con endpoints REST
7. Habilitar CrossOrigin en los controladores

## Endpoints
| Metodo | URL | Descripcion |
|--------|-----|-------------|
| GET | /api/cliente | Listar clientes |
| GET | /api/cliente/{id} | Obtener cliente |
| POST | /api/cliente | Crear cliente |
| PUT | /api/cliente/{id} | Actualizar cliente |
| DELETE | /api/cliente/{id} | Eliminar cliente |
| GET | /api/producto | Listar productos |
| GET | /api/producto/{id} | Obtener producto |
| POST | /api/producto | Crear producto |
| PUT | /api/producto/{id} | Actualizar producto |
| DELETE | /api/producto/{id} | Eliminar producto |
