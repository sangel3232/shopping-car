# HU-01 - Configuracion inicial

## Informacion General
- ID: HU-01 | Sprint: 1 | Estado: Completada

## Historia de Usuario
Como desarrollador, quiero la estructura base con Spring Boot y PostgreSQL,
para comenzar a desarrollar el sistema de carrito de compras.

## Criterios de Aceptacion
- Proyecto Spring Boot 3.2 inicia sin errores
- Conexion a PostgreSQL establecida
- Base de datos distribuidos accesible
- Tablas creadas con ddl-auto=update
- Servidor en puerto 9000

## Configuracion
```properties
server.port=9000
spring.datasource.url=jdbc:postgresql://127.0.0.1:5432/distribuidos
spring.datasource.username=postgres
spring.datasource.password=1234
spring.jpa.hibernate.ddl-auto=update
```
